# Hotel Management System

## Project Overview

The Hotel Management System is a Java-based application developed to manage hotel operations efficiently. 
The system helps manage room bookings, customer details, employee information, and billing processes.

This project provides an easy-to-use interface for hotel administrators to maintain records and perform daily hotel management tasks.

## Features

- Customer registration and management
- Room booking and cancellation
- Room availability checking
- Employee management
- Billing and payment management
- Database integration for storing records
- User-friendly interface

## Technologies Used

- Java
- Java Swing
- JDBC
- MySQL Database
